Spoken languages

A visualization about the number of languages per country. A grayscale colorscheme has been chosen because this is the easiest to perceive. Data is missing for 1 country (Somaliland).

This map enables for two tasks.
One: the comparison of the number of official languages per country. This is done by using a grayscale colorscale. More appropriate would be a discrete scale to get a more even distribution of the color, but did not get that working (yet). Furthermore it would be more appropriate to only highlight the borders of a acountry, instead of also changing the fillcolor, but also did not get that working too.
Two: Inquiry about the number of languages per specific country. This is done by a mousehover with a popup that shows both the number of languages and which languages are spoken.
Extra: it is possible to zoom in, enabling the inspection of areas with small countries.
